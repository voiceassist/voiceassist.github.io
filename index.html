List of Available Commands

Weather - Gets the weather based on the query

  • "Weather { any city name }"

  • "Weather { any country name }"

  • "Weather { any postal code }"

Soccer/Football Information - Gets relevant soccer information

  • "Soccer/Football standings { League codes }"

      • List of "League codes"

    • "PL" - English Premier League

    • "EL1" - English League One

    • "FAC" - FA Cup

    • "BL1" - Bundesliga

    • "FL1" - Ligue 1

    • "SA" - Serie A

    • "PD" - La Liga

  • "Soccer/Football fixtures { Team Name }" - Team name can be replaced by any major soccer club

  • "Soccer/Football fixtures next/previous { number } { Team Name }" - Gets upcoming/past "x" fixtures of a particular team

  • "Soccer scores live" - Get live scores of teams playing in the major European leagues

Playing Songs

  • Play { Song Name } - Provides a spotify link as well for any song

  • Play { Song name } by {Artist name }

Places of interest

  • Show me { places } near my location

  • Show me { places } near my { address } - Any complete address eg: 181 Lester St, Waterloo, ON

      • List of available "places"

    • Resturants

    • Hospitals

    • Stores

    • Cinemas

    • Subway

Navigation

  • Directions from { From address } to { To address } by { transport modes }

      • List of "transport modes" available

    • Car/Driving

    • Bus

    • Walk

    • Cycle/Bike

Generic Search Command

  • Search { Anything } - Returns a google search link of any text spoken after "Search"
